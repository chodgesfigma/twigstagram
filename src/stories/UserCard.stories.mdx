# UserCard Component

The `UserCard` component is used to display user information in the Twigstagram application. It includes the user's profile picture, name, username, and a follow button.

## Example

Here is an example of how the `UserCard` component looks and functions:

<Canvas>
  <Story name="Default">
    {() => (
      <QueryClientProvider client={new QueryClient()}>
        <MemoryRouter>
          <UserCard 
            user={{
              $id: 'user_id',
              imageUrl: '/assets/icons/profile-placeholder.svg', // Example profile image URL
              name: 'John Doe',
              username: 'johndoe',
            }} 
          />
        </MemoryRouter>
      </QueryClientProvider>
    )}
  </Story>
</Canvas>

## Usage

To use the `UserCard` component in your project, you can import it and use it with the desired props. Below is an example of how to include it in your application.

```jsx
import { Models } from "appwrite";
import { Link } from "react-router-dom";

import { Button } from "../ui/button";

type UserCardProps = {
  user: Models.Document;
};

const UserCard = ({ user }: UserCardProps) => {
  return (
    <Link to={`/profile/${user.$id}`} className="user-card">
      <img
        src={user.imageUrl || "/assets/icons/profile-placeholder.svg"}
        alt="creator"
        className="rounded-full w-14 h-14"
      />

      <div className="flex-center flex-col gap-1">
        <p className="base-medium text-light-1 text-center line-clamp-1">
          {user.name}
        </p>
        <p className="small-regular text-light-3 text-center line-clamp-1">
          @{user.username}
        </p>
      </div>

      <Button type="button" size="sm" className="shad-button_primary px-5">
        Follow
      </Button>
    </Link>
  );
};

export default UserCard;
```
